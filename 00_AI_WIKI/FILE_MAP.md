## 新对话冷启动流程

对 Claude 说：
> 读 story_system/README.md，然后从当前章继续。

Claude 会按此顺序加载：
1. `README.md` → 知道项目是什么
2. `章节进度表.md` → 知道写到哪了
3. `伏笔清单.md` → 知道有什么伏笔
4. `人物状态.md` → 知道每个人当前状态
5. `写作风格指南.md` → 知道怎么写
6. 当期章节大纲 → 开始写作
