# Repository Publication Boundary

This repository, `FeiYun-Novel/webnovel-writing-system`, is the sole public release repository for the generic writing-system architecture.

It may contain:

- generic workflows and checklists
- empty templates and abstract examples
- scripts that operate on user-supplied project files
- public documentation required to understand the architecture

It must not contain:

- novel prose, drafts, chapter summaries, or outlines
- characters, plot, setting, foreshadowing, or live story-state records
- private technique corpora, images, archives, or local troubleshooting excerpts
- local absolute paths, real personal email addresses, credentials, tokens, cookies, or session logs

The full working novel project is maintained separately in a private repository. The two repositories are not interchangeable, and changing repository visibility is not a release workflow.
