# 阶段 4 检查表 · 写后审稿

> 本页只保留阶段入口，避免审稿规则出现两份权威。完整规则见 [`跨模型写后审稿SOP.md`](./跨模型写后审稿SOP.md)。

- [ ] 完整初稿已经保存，不审半章或零散片段。
- [ ] 机械机检已通过；脚本能判断的项目不再浪费模型复查。
- [ ] 已填写本章触碰清单和风险标签。
- [ ] 审稿模型提供方不同于主笔提供方。
- [ ] 固定运行“读者体验”和“写法与 AI 痕迹”，其他路线按风险启动。
- [ ] 各责任项已经认领，主笔只补没有被脚本或审稿路线覆盖的空缺。
- [ ] 审稿意见带位置、证据、等级和最小修改建议，并按根因去重。
- [ ] 主笔统一修订，审稿模型不直接接管正文。
- [ ] 按修改规模执行局部复核、受影响路线复核或整章重审。
- [ ] 正式审稿最多两轮；仍有争议时停止循环并交用户裁决。
- [ ] 全部适用检查通过后只标记为候选定稿；用户认可并完成阶段 5 后才正式封存。
