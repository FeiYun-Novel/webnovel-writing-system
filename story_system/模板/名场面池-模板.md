# 名场面池（模板）

> **用途：** 全书预定要写的标志性桥段/名场面银行——写前先来查一下，别在铺垫不够的情况下提前把名场面写崩，也别漏埋该有的高光时刻。
> **对应项目文件：** 通常放在 `story_system/名场面池.md`，与分卷大纲、伏笔清单配合使用。
> **使用协议：** 想到一个名场面点子 → 新增一行，状态标"未写"；写到触发条件成熟时才动笔；写完后回来标"已写第X章"并简单复盘有没有写到位。

## 名场面清单

| 场面名称 | 触发条件（写到什么程度才解锁） | 涉及角色 | 预期效果（燃/虐/爽/其他） | 状态 | 备注（别提前写崩的注意点） |
|----------|-------------------------------|----------|---------------------------|------|---------------------------|
| [示例：身份暴露瞬间] | [示例：主角伪装已维持到临界点，且关键角色已积累足够怀疑] | [示例：主角、感情线角色一] | [示例：爽+虐] | 未写 | [示例：铺垫不够就提前写会显得突兀，需先埋够怀疑线索] |
| [场面名称] | | | | 未写 | |
