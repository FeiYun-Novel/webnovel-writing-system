# 示例：把通用槽位填成一个具体项目

本系统所有文件用的都是**通用角色槽位**（主角 / 反派 / 感情线角色 / 导师NPC / 势力…），任何小说都能对号入座。这份文档用**抽象示例**演示这些槽位该怎么填成具体设定——**仅供参考，不是使用前提**，下面的填法是抽象举例，不指向任何具体作品，你可以完全换成自己的题材。

## 槽位 → 具体设定（抽象示例映射）

| 通用槽位 | 抽象示例填法 |
|---|---|
| 题材 | 题材A + 核心机制B（你可以换成都市 / 奇幻 / 悬疑 / 古言等任意组合，不必和下面举例一致） |
| `主角` | 一个因某种契机被卷入某个身份或立场的普通人 |
| `反派 / 对手` | 与主角立场相对的另一位核心角色 |
| `感情线角色一/二/三…` | 若干位与主角产生羁绊的角色（数量、性别、关系类型均可自定） |
| `导师/长辈NPC` | 给主角指引或施加压力的长辈/上级角色 |
| `势力/组织` | 主角所处的组织或阵营 |

## 可选机制槽位（按题材开关）

这几个是**可选**的——你的小说没有就跳过：

### 1. `特殊能力 / 金手指`（如果你的题材需要）
抽象示例：主角拥有一种别人没有的能力、工具或提示机制。可以是任何形式——一项天赋、一件外物、一套系统提示，也可以完全没有。

### 2. `主角专属信息机制`
抽象示例：主角掌握着别人不知道的信息（可能是身份、可能是内心想法、也可能是对未来的预知），这类"信息差"是很多题材制造张力/喜剧/悬念的常见手法。
你可以换成：预知、读心、面板信息、弹幕、旁白只有主角能看、内心独白被部分角色听到等任何"信息不对称"设定。

> 这类机制往往需要一套**信息可见性约定**：谁能听到什么、用什么符号区分。这个**约定设计的思路**是通用的，符号和规则你自己定。

### 3. `角色专属反应特征`
抽象示例：某个角色有一种**专属的**身体/神态反应特征，情绪波动时通过它泄露真实心理（正文可以只写形状质感与动作神态，不直接点破具体隐喻）。
你可以换成：口头禅、习惯性小动作、瞳色变化、耳根发红、握拳、语速变化等任何「一角色一签名反应」。

## 要点

- **通用骨架**（流程 / 检查表 / 技法库 / 上下文管理 / 跨提供方审稿）拿来即用，和题材无关。
- **具体填充**（人物 / 设定 / 题材 / 专属机制）是你自己的创作，用空白模板（`templates/`、`story_system/模板/`）建立即可。
- 别被上面的抽象举例带跑——那只是演示槽位怎么填，不对应任何具体作品的设定组合。

## 最小完整样例（照着抄"填到什么程度"）

> 下面是一个**纯虚构的演示项目**，和任何真实作品无关，只为让你看清"每个槽位填到多具体就能开写"。**别照抄内容，只照抄"填得多细"这个程度。** 题材随便挑的，你换成自己的即可。

### 演示题材

> 都市 + 异能 + 悬疑轻探案。核心机制：主角能"读到物品上残留的最后一段强烈情绪"。

### 角色卡样例 → `角色/主角.md`

```markdown
# 陆时 · 主角

- **定位：** 老城区一家旧当铺的年轻老板，被卷进一桩连环失窃案。
- **来历：** 家传当铺第三代，父亲失踪那年他继承了铺子，也第一次发现自己能"读物"。
- **性格：** 表面懒散、爱装穷；实则记性极好、认死理，见不得旁人被冤枉。
- **他想要什么（核心驱动）：** 查清父亲失踪真相；短期目标是先摘掉自己"销赃嫌疑人"的帽子。
- **能力与代价：** 手触物品能看到它最后一位持有者的强烈情绪画面；读一次头痛半天，情绪太烈会短暂失明。
- **说话腔调：** 半句正经半句贫，被戳中痛处会突然沉默。
- **一个标志性反应：** 紧张时会无意识摩挲柜台上那枚旧铜钱。
```

### 世界观样例 → `设定/世界观与体系.md`

```markdown
## 基调
现实都市，异能是极少数人才有的隐秘天赋，不为大众所知。

## 能力体系
- 「读物者」：能从物品读取残留情绪/画面，强弱按"共感深度"分三档。
- 晋升逻辑：靠真实经历的情绪冲击"开档"，不能刻意练。
- 上限：只能读"最后一位持有者"、只读情绪最强的那一段。

## 核心规则与限制（硬设定，不可违反）
- 读到的是"情绪与画面碎片"，不是完整事实——会误导，也会成为反转来源。
- 读物有代价（头痛/短暂失明），不能连续滥用。
- 异能圈子有默契：不公开、不滥用于牟利，违者会被"清理"。
```

### 第一章计划样例（阶段2 产出的东西长这样）

```markdown
- **本章目标：** 让主角登场 + 抛出"他被列为销赃嫌疑人"的危机钩子。
- **核心冲突：** 一件失窃的古董出现在他当铺，警方找上门，他百口莫辩。
- **场景列表：**
  1. 当铺日常 + 用一次异能，立"读物"人设（前300字给抓手）
  2. 刑警上门，古董=赃物，冲突爆发
  3. 他偷偷读那件古董，看到一段不属于"小偷"的诡异情绪 → 疑点
- **伏笔操作：** 埋"那段情绪不对劲"（F01）；埋父亲旧铜钱（F02）。
- **章末钩子：** 他认出画面里一闪而过的人，是三年前失踪的父亲。
- **技法清单：** 场景1→白描（开铺子的动作抓手）；场景2→对话（警察施压的张力）；场景3→六觉/心理（读物时的通感与眩晕）。
```

> 看到了吧——**填到这个程度（每人卡片一屏、世界观半屏、第一章计划一屏）就足够让 AI 开写第一章了。** 其余的边写边补：技法库先填 3-5 份、风格规则写几章后再固化。
